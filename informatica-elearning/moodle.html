What is Moodle?

  • Moodle is a free learning management system
  • Moodle can be used in many types of environments such as in education, training and development.

Moodle features

Typical features of Moodle are:

  • Assignment submission
  • Discussion forum
  • Files download
  • Grading
  • On-line quiz

Moodle plug-ins

Moodle's infrastructure supports many types of plug-ins such as:

  • activities (including word and maths games),
  • question types (multiple choice, true and false, fill in the blank),
  • graphical themes.

Interoperability

  • Moodle runs without modification on Unix, Linux, Windows, Mac OS X and any other systems that support PHP and a database,
  • data goes in a single database such as MySQL or PostgreSQL.

Acronyms

  • LMS: learning management system.
  • HR: Human resources.
  • Moodle: Modular Object-Oriented Dynamic Learning Environment.
  • MySQL
